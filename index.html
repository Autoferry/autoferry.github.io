This page presents project- and master thesis proposals for the Autosea project. The project have scheduled several experiments, starting in December. This means that many of the results from the suggested theses can be verified with real-world data, or tested in an experiment. The project software is developed using ROS, which means that features can be implemented in C++ or Python. Additionally, a simulator is being developed in Gazebo.

General information about the project can be found at AMOS' website.
