This page presents project- and master thesis proposals for the Autoferry project. Many of the results from the suggested theses can be verified with the gathered data, or tested in new experiments. Matlab and Python are the main tools used for development and simulation. Some project software is developed using ROS

General information about the project can be found at Edmund Brekke's website.
